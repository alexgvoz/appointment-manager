# Appointment Manager Application

Application for managing appointments and customers. Connects to a database.

## Program Info
Version: 1.0
IDE: IntelliJ IDEA 2022.1.3
JDK: Java SE 17.0.1
JavaFX: JavaFX-SDK-17.0.1
MySQL: mysql-connector-java-8.0.25

## Launching Directions
Open up the project folder in IntelliJ IDEA.
Inside IntelliJ, right click on AppointmentManager in src > main > java > com.example.appointmentmanager.
Select "Run 'AppointmentMan...main()' " from the list.

## Additional Report Info
On the reports page, I decided to provide addition reports on total number of appointments based on countries and another report based on customers.
